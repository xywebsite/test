Web Development

Introduction to Web Development, Html, Css.

Today, I learnt about the basics of Web Development, Html, Css.

I learnt about the importance of documenting work and listing down what I learnt every week.
We were then introduced to pure text editors, such as Sublime Text and Visual Studio Code.
Personally, I prefer Sublime Text as I’m more familiar with it.
The advantage it had over other text editors is that it is Open Source, and Free to use.
It also has unlimited text length and it is cross platform.

Introducing HTML, the standard markup language for creating Web pages.
HTML describes the structure of a Web page , and HTML consists of a series of elements, a start and end tag.
Each element contains content. Elements tell the browser how to display the content, HTML defines the content.

I was then tasked to make a simple sample website using HTML.
Firstly, I created a folder “mywebsite” then I created a file “index.html” within the folder.
Then I edited and entered the contents of a basic web-page, headings, paragraphs, lists and added pictures.
Then I saved my file and displayed it on the browser to check if it came out right.

I also learnt about CSS, which is used for beautifying and presenting layout information.
I then watched the Jake Wright - “Learn CSS in 12 minutes” video on YouTube.
It shows the creation of a site content, and it shows how an effective layout can be planned.
Jake Wright uses id (e.g. #container, #navigation) to define his rules for his page.
